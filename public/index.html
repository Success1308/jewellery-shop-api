Welcome to the Jewelry Shop API

Overview:

This API provides information about our jewelry collection, including items for men and women.

Get All Jewelry Items

Get a list of all jewelry items available in the shop.

Endpoint: /jewelry

Get Jewelry by ID

Get information about a specific jewelry item by its ID.

Endpoint: /jewelry/{id} (replace `{id}` with a number)

Example Usage:

To get all jewelry items:

GET /jewelry

To get jewelry with ID 1:

GET /jewelry/1
